# 📰 News Portal — Version 2

An upgraded version of my original news portal project, featuring a refined layout, improved JavaScript architecture, and cleaner overall UI. Built with vanilla HTML, CSS, and JavaScript.

## 🔍 About

News Portal 2 is a direct evolution of [News-Portal-1](https://github.com/Waish228/News-Portal-1). The structure was reorganized, the CSS was rewritten for better consistency, and the JavaScript was split into logical modules for cleaner separation of concerns.

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Waish228/News-Portal-2.git
   ```
2. Open `News Portal 2 Home pages.html` in your browser.

No build tools or installations required.

## 🆚 Differences from V1

- Cleaner CSS structure with more consistent spacing and typography
- Improved JavaScript organization across two logical files
- Better overall page layout and readability
